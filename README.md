# pelmeneva-front

## About project

** This is client side of "Pelmen Eva" site **

## ⚙️ Launch

- **Download Node.js (version >= 22)**

- **Download zip archive with project**

- **In a project folder**
  ```
  yarn
  yarn build
  yarn start

  ```
  
## ⚙️ Launch in mock mode

- **Build mockServiceWorker.js**
  ```
  yarn build:mock

  ```

- **Run app with mock tests**
  ```
  yarn start:mock

  ```
